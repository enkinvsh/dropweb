# Заголовки ответа Remnawave

Заголовки ответа можно настроить в Remnawave как custom response headers для хоста подписки. Хэдеры будут отправлены вместе с содержимым подписки.

Dropweb читает эти HTTP response headers при загрузке подписки и использует их для включения дополнительных действий в приложении.

| Заголовок | Значение | Что делает Dropweb |
|---|---|---|
| `dropweb-cabinet` | HTTPS URL личного кабинета, например `https://cab.example.com` | Показывает действие личного кабинета и открывает указанный URL в браузере или custom tab. |

Требования к `dropweb-cabinet`:

- значение должно быть полным HTTPS URL;
- заголовок должен приходить в ответе на запрос подписки вместе с ее содержимым;
- если заголовка нет, Dropweb не показывает действие личного кабинета.
