# Режим разработчика

Скрытый режим с расширенными настройками. По умолчанию выключен.

## Как включить

1. Откройте меню карточки на дашборде (свайп снизу-вверх или тап по стрелке-подсказке) и выберите **Настройки**.
2. На открывшемся экране **«Настройки»** сделайте **5 быстрых тапов по заголовку** экрана — пять подряд в течение ~3 секунд. Если тапать медленнее, счётчик сбрасывается.
3. Появится уведомление о включении — режим разработчика активен. Настройка сохраняется между запусками.

## Что появляется

После включения в настройках открываются расширенные пункты:

- **Контроль доступа** (Access Control) — какие приложения идут через VPN;
- **Конфиг** (Config) — редактор конфигурации ядра;
- **Настройки приложения** — низкоуровневые параметры клиента.

> Режим для продвинутых пользователей: неаккуратная правка конфига может нарушить подключение.
